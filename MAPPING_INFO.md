## 映射关系说明

每个映射包含：
1. **A股代码**: 6位数字股票代码
2. **公司名称**: A股公司中文名称
3. **关联关系**: 说明该A股公司与美股公司的业务关联

## 映射逻辑

映射基于以下维度：
- 📦 **供应链关系**: 如立讯精密是苹果的供应商
- 🏭 **同行业竞争**: 如比亚迪与特斯拉都是电动车制造商
- 🔧 **技术相似**: 如科大讯飞与微软都涉及AI技术
- 📊 **业务对标**: 如用友网络对标Salesforce的企业软件

## 如何添加新映射

编辑 `lib/stock-mapping-data.ts` 文件，按以下格式添加：

```typescript
'SYMBOL': {
  symbol: 'SYMBOL',
  business: '主营业务描述',
  aStockLeaders: [
    { code: '股票代码', name: '公司名称', relation: '关联关系' },
    // ... 最多4个
  ]
}
```

## 注意事项

1. 映射关系仅供参考，不构成投资建议
2. A股公司可能只是部分业务与美股公司相关
3. 映射会随着公司业务变化而需要更新
4. 未在映射表中的美股公司会显示"暂无业务描述"
