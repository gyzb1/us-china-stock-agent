# 快速开始指南

## 1. 启动开发服务器

无需配置API Key，直接启动即可！

```bash
npm run dev
```

## 3. 访问应用

打开浏览器访问: http://localhost:3000

## 功能说明

### 刷新数据按钮
- 快速获取最新的股票行情数据
- 显示成交额排名前15的美股公司（已过滤ETF）
- 响应速度快（1-2秒）

### 获取新闻按钮
- 获取每只股票的东财最新资讯
- 自动生成新闻摘要
- 响应快速（5-10秒），无需等待

## 数据来源

- **股票行情**: 东方财富API
- **新闻资讯**: 东方财富API
- **完全免费**: 无需API Key

## 优势特点

1. ✅ 无需注册和API Key
2. ✅ 完全免费，无调用限制
3. ✅ 响应速度快
4. ✅ 数据来源统一，更可靠
5. ✅ 包含中美股票映射关系

## API 端点

- `GET /api/stocks` - 获取股票行情
- `GET /api/news?symbol=AAPL` - 获取单只股票新闻
- `GET /api/stocks-with-news?limit=5` - 获取股票+新闻（慢）

## 项目特点

## 卡片信息说明

每个股票卡片包含：
- 📈 **实时行情**: 股价、涨跌幅、成交额、成交量
- 💼 **主营业务**: 公司核心业务描述
- 🔗 **A股映射**: 对应的A股龙头公司（股票代码、名称、关联关系）
- 📰 **新闻摘要**: 最新新闻和市场情绪分析（需要API Key）
