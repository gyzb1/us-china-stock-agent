# 新闻质量优化

## 🎯 问题分析

### 发现的问题

1. **显示了10/28的旧新闻**
   - 时间过滤没有生效
   - 原因：获取的新闻数量太少，过滤后为空

2. **没找到重要新闻**
   - 应该显示："存储芯片正迎来前所未遇繁荣期"
   - 实际显示：不相关的旧新闻

3. **AI总结太简单**
   - 没有抓住"爆表业绩"、"繁荣期"等关键词
   - 缺少具体数据和趋势分析

---

## ✅ 解决方案

### 1. 增加新闻获取数量

**改进**：
- 从获取5条 → 20条
- 过滤后再返回5条
- 确保有足够的新闻可供筛选

**代码**：
```typescript
pageSize: 20, // 增加获取数量，后面再过滤
```

---

### 2. 放宽时间限制策略

**改进**：
- 优先获取近2天的新闻
- 如果近2天没有 → 放宽到5天
- 避免因为太严格而没有新闻

**逻辑**：
```typescript
// 先尝试过滤近2天的新闻
let recentNews = filterRecentNews(allNews, 2);

// 如果近2天没有新闻，放宽到5天
if (recentNews.length === 0) {
  console.log(`No news in 2 days for ${symbol}, trying 5 days...`);
  recentNews = filterRecentNews(allNews, 5);
}
```

---

### 3. 改进搜索关键词

**改进**：为每个公司添加多个搜索词

**MU的搜索词**：
```typescript
'MU': ['Micron 美光', '存储芯片', 'DRAM', 'NAND']
```

**优势**：
- 中英文结合
- 包含行业关键词
- 更容易找到相关新闻

---

### 4. 优化AI提示词

**改进重点**：

#### 重点关注的标志词
```
1. 爆表业绩
2. 繁荣期
3. 历史新高
4. 大幅增长
5. 突破性进展
```

#### 关注的内容
```
1. 价格趋势：涨价、降价、价格周期
2. 财报数据：营收、利润、增长率、超预期
3. 行业趋势：供需关系、市场格局
4. 华尔街观点：分析师评级、机构看法
```

#### 输出要求
```
- 80-120字简洁中文
- 突出最重要的1-2个事件
- 包含具体数据
- 分析对股价的影响
```

#### 示例输出
```
存储芯片迎来前所未有繁荣期，DRAM和NAND价格持续
上涨15-20%。美光Q3业绩爆表，营收增长34%超预期。
华尔街分析师上调目标价，预计涨价周期将持续至明年Q2。
```

✅ 爆表业绩

✅ 超预期
