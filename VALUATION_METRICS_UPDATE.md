## 📊 包含的估值指标

### 核心指标

1. **PE (市盈率)**
   - PE(TTM)：滚动市盈率
   - PE(静态)：静态市盈率
   - 说明：股价/每股收益，衡量估值高低

2. **PB (市净率)**
   - 股价/每股净资产
   - 衡量资产价值

3. **ROE (净资产收益率)**
   - 净利润/净资产
   - 衡量盈利能力

4. **估值水平评价**
   - 相对行业平均水平
   - 高估/合理/低估

```

1. 必须包含PE、PB、ROE等核心指标

4. 说明盈利能力
5. 如有特殊情况说明（如龙头溢价、周期底部等）
```

## 📊 不同行业的估值特点

### 科技股

**特点**：
- PE通常较高（20-40倍）
- PB较高（3-8倍）
- ROE中等（15-25%）
- 看重成长性

**示例**：
```
PE(TTM) 30倍，PB 5.5倍，ROE 20%，估值合理，成长性支撑估值
```

### 金融股

**特点**：
- PE较低（5-12倍）
- PB较低（0.8-1.5倍）
- ROE稳定（12-18%）
- 看重股息率

**示例**：
```
PE(TTM) 8倍，PB 1.1倍，ROE 16%，估值较低，股息率4.5%，价值投资标的
```

### 消费股

**特点**：
- PE中等（15-30倍）
- PB中等（2-5倍）
- ROE较高（18-30%）
- 看重品牌和渠道

**示例**：
```
PE(TTM) 25倍，PB 4.0倍，ROE 25%，估值合理，品牌溢价明显，盈利能力强
```

### 周期股

**特点**：
- PE波动大（5-50倍）
- PB较低（1-3倍）
- ROE波动大（5-30%）
- 看重周期位置

**示例**：
```
PE(TTM) 12倍，PB 1.8倍，ROE 10%，处于周期底部，估值合理，待复苏
```
